# 浏览器端 PDF 合并工具（按上传顺序合并）

这是一个纯前端的 PDF 合并小工具，使用 pdf-lib 在浏览器内合并多个 PDF 文件，并按用户上传/列表顺序输出合并后的 PDF 文件，免服务器部署。

特点
- 纯前端实现：不上传文件到服务器，隐私更安全。
- 保留上传顺序：默认按选择顺序合并，支持上移/下移调整顺序。
- 支持批量选择多个 PDF，合并后自动下载 merged.pdf。

使用方法
1. 在文件管理器中打开或把整个项目目录放在静态服务器上；也可以直接用浏览器打开 `index.html`（大多数现代浏览器允许直接运行）。
2. 点击“选择文件”或将文件拖入（注：本示例未实现拖拽，但可通过文件选择）并选择多个 PDF。
3. 在文件列表中可使用上移/下移按钮调整文件合并顺序，或删除不需要的文件。
4. 点击“合并并下载”，浏览器将生成并下载合并后的 `merged.pdf`。

兼容性
- 推荐使用最新版本的 Chrome、Edge、Firefox 或 Safari。pdf-lib 在这些浏览器上工作良好。
- 文件较大或页数较多时，合并会占用一定内存，可能导致处理变慢或浏览器卡顿。

安全与隐私
- 所有文件处理都在本地浏览器完成，文件不会上传到第三方服务器。

常见问题
- 如果点击合并没有反应：检查浏览器控制台是否有错误（可能是网络加载 pdf-lib CDN 失败）。
- 想在局域网或远程访问：在项目目录启动静态服务器（例如 `python -m http.server 8000`），然后在其他设备浏览器打开 `http://<host>:8000`。
